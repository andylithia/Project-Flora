简介
包装内容
1. 1x 底座零件包
2. 4x “Kartusche” 真空管模块
3. 1x Fujitsu USB电源
4. 1x MiniUSB 连接线

特性
Numitron电阻丝式真空管
该类型真空管为70年代的产物，其结构类似于电灯。优点为工作电压非常低，在断开电源后因为灯丝仍然有热度，可以保持发光短暂时间。因而适合高振动场合。 Project Flora使用的为苏联Reflector/ Sovtec生产的IV-9 (ИВ-9)，为微型7段式带小数点显示器。使用寿命极长，在良好的工作环境下可连续工作超过10000小时。

模块化设计
Project Flora中，真空管与其背景光效LED集成在了一个模组中，称为“Kartusche”。除了简化了安装过程，该设计使得任何部分出现故障时，可以减少更换修复的成本。

可配置光效
Project Flora具有丰富的光效功能，可以任意合成RGB背景光，调节每一枚Numitron真空管的亮度。在主界面下用户可以快速切换几个预先配置好的光效。该设计使得用户可以在休息时间方便地切换到低亮度模式，然后在醒来后换回正常亮度。

极高精度实时时钟芯片
Project Flora使用了DS3231这枚内置温度补偿晶体的极高精度实时时钟集成芯片。在0-40°C的室温条件下稳定性可达±2ppm。使用了一枚纽扣电池作为备份电源，使得在主电源切断时亦可持续走时，避免了多次调整时间的麻烦。

DFU功能
Project Flora使用了内置USB控制器的STM32F0系列高性能控制器。该控制器可以运行设备固件升级程序（Device Firmware Update, DFU），进行固件升级以增加新功能。详情参见本文档的DFU相关内容。

组装
为了避免运输中损坏，该时钟设计为套件形式，使用前需要简单的组装。

零件包中含有以下内容：
1. 1x “Base” 主板
2. 1x 铝合金外壳（磨砂或抛光型）
3. 4x 0.8mm尼龙垫片
4. 4x 平头M3螺丝
5. 4x 集成垫片的圆头M3螺丝
6. 1x 亚克力背板
7. 1x 亚克力底板
8. 8x 黑色硅胶自粘脚垫


第一步.
将垫片摆放在铝合金外壳内如图所示的位置，如有必要可以使用少量胶水辅助固定。

第二步.
撕下亚克力背板的保护膜，将主板与亚克力背板一同放入铝合金外壳。
注意：电子部分易受静电损害，在进行组装前请对身体进行放电（如触摸接地的电器外壳），操作过程中避免直接触摸电路部分，尽可能仅接触电路板边缘。
注意：为避免意外情况下电子部分与金属外壳相接触，其表面贴有一些保护胶带，请勿将其移除。

第三步.
使用平头M3螺丝将其固定，注意该步骤完成后，螺丝顶部应当与铝合金外壳内的结构平齐。

第四步.
撕下亚克力底板的保护膜，安装亚克力底板，在合适的位置粘贴四个脚垫。

第五步.
安装四枚Kartusche模块
注意：为避免损伤真空管，安装时请确保受力均匀

第六步.
插入Mini-USB Type-A线，观察显示组件自检是否正常
如果安装正确，背光应当显示橘色，显示内容为四个轮流亮起-淡出的数字'8'

至此组装工作完成，短按滚轮显示欢迎画面

使用

操作方式
该时钟接受下列四个操作：向左滚轮、向右滚轮、短按滚轮、长按滚轮
通常意义下，向左滚轮意味着“上一项”、“减少”，向右滚轮意味着“下一项”、“增加”。
长按通常为1秒左右，按键超过三秒会被当作无效（取消操作）

除此之外背面还有两个隐藏式按键，在需要使用的情形下使用细棒状物以接触。
从背面看，左按键为“BOOT0”，在上电时保持该按键按下，可进入DFU固件升级模式（参考后文相关内容）；右按键为“NRST”，在运行时按下该按键可进行复位操作，与重新上电效果相同。

字体
该时钟具有以下两个字体，区别在对英文字母的处理方式上。
第一个字体具有完整的6bit ASCII字符，但是不熟悉的情况下比较难以辨识；
第二个字体可阅读性较强，但是仅包括部分字符，部分字符具有歧义。


Project-Flora使用的操作系统为事件响应式的LSYS。
在接入电源开机后，首先显示的为自检界面，按下滚轮进入下一个界面
第二个显示的界面为欢迎信息，mua♡，显示内容在显示完成后会重新开始，按下滚轮进入主界面

主界面包括以下内容，滚动滚轮以切换，在切换时会短暂显示下一项的编号和名称（3个英文字符）：
1. 时间		TIM	TIMe
2. 日历		DAT	DATe
3. 计时		CST	Countdown, Stopwatch, Timer
4. 系统设置	SCF	System ConFig

前三个界面中，短按可以触发一些特殊功能，长按进入设置画面，
以时间为例，短按进入快速光效设置，长按进入时间设置。
详细操作参考下图，出于时间有限，第一次发布的软件中可能有部分功能未实现。

维护

因为本设备并非封闭式设计，避免用含水量过高的纸张擦拭。
